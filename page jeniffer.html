Retouches de photographie (Jeniffer)

Que ce soit pour supprimer un élément indésirable sur une photo, corriger des défauts comme la luminosité, la perspective ou autres ou tout simplement pour embellir une photo, le logiciel GIMP est fait pour ce type de retouches. Semblable à Photoshop, ce logiciel compte avec plusieurs outils de retouche photo afin que l'utilisateur puisse obtenir le résultat désiré.

Transformation

Pour commencer, il est possible de redresser une image à l'aide de l'outil rotation. Dans la boîte à outils, en choisissant l'outil rotation il est possible de définir l'angle dans lequel on veut que notre image soit dirigé ou afin d'obtenir une image droite ainsi que de lui donner une nouvelle perspective si on le désire. Cet outil est donc utilisé afin de rendre la photo/image droite ou dans l'angle désiré, cependant il y aura des espaces vides qu'on va pouvoir remplir ou bien supprimer plus tard. De plus, il y a d'autres options afin de découper l'image, changer sa perspective et autres.

Pour ce faire il suffit de cliquer sur l'onglet outils et ensuite dans l'option transformations et choisir celle qui vous convient

Couleurs

Il est possible de corriger les couleurs d'une photo ou tout simplement de les changer. Le logiciel a des couleurs prédéfinis que l'on peut sélectionner :

Couleurs prédéfinis
Normaliser L'outil normaliser est utilisé afin de rendre tous les couleurs de façon qu'ils aient le même teint. Les couleurs sombres sont le plus sombre possible ainsi que les couleurs clairs sont le plus clairs possible.
Égaliser L'outil égaliser est utilisé afin de rendre toutes les couleurs presque la même couleur.
Renforcer les couleurs L'outil renforcer les couleurs est utilisé afin d'augmenter la saturation de l'image, sans modifier la luminosité de celle-ci.
Étendre TSV L'outil étendre le contraste agit afin de canaliser les couleurs bleu, rouge et vert sans modifier la luminosité. Tout ce, afin qu'aucune couleur soit prédominante.
Balance des blancs L'outil balance des blancs sert à l'amélioration des couleurs noirs et blancs et ainsi balance aussi les autres couleurs.

Pour ce faire il suffit de cliquer sur l'onglet couleurs auto et choisir celui qui pourrait vous convenir.

Dans le cas où on voudrait ajuster nous même les couleurs prédominantes de notre image il est possible en choisissant l'option teinte et saturation, toujours dans l'onglet couleurs, qui nous permet aussi d'ajuster la luminosité.

Il y a aussi la possibilité de changer les couleurs de l'image à l'aide de l'option courbes , dans l'onglet couleurs.

Filtres

GIMP offre plusieurs filtres afin d'améliorer les images. Ces filtres ne sont pas seulement utilisés pour l'amélioration des images mais aussi afin de donner des effets spéciaux à l'image. Le plus connu étant le filtre pour supprimer les yeux rouges.

Pour ce faire cliquer sur l'onglet filtres

Filtres
Flou Filtre afin de donner du flou à une image ou à une partie de l'image.
Amélioration Filtres utilisé pour améliorer l'image et attenuer les imperfections.
Distorsion Effet pour transformer l'image (Déformation, mosa&itremque
Ombres et lumières Effet d'éblouissement ou éclairage.
Génériques Création de filtres.
Combiner Associer deux images ou plus.
Artistiques Effet artistique à une image. (Dessin,cubisme,etc)
Décor Bordures décoratives et effet spéciaux.
Web Filtres utilisés pour les images destinés au web. (Carte cliquable)
Animation Filtres pour la création de GIF.

Autres

Afin de supprimer un élément indésirable d'une image il est possible de le faire à l'aide de l'outil clonage. Il suffit de choisir une partie de l'image qui pourrait remplacer l'élément sans que la correction paraisse.
