Manipulation de texte

Pour créer du texte, il faut utiliser l'outil de texte.

Il est possible de déplacer une boite de texte avec l'outil déplacement, mais il faut cliquer sur une des lettres et non sur l'arrière plan.

Il est possible de créer des caractères Unicode spéciaux en sélectionnant une boite de texte et en pesant sur CTRL + MAJ + U . Il faut ensuite entrer le code Unicode du caractère voulu.

Les dégradés

Outil de dégradé

Options :
Opacité : Change la vivacité des couleurs Type de dégradé : Changes les couleurs du dégradés Forme : Change le motif que le dégradé effectue Répetition : Forme de répétition effectué dans le motif

Créer un dégrader dans une sélection:
Faire une sélection. Choisir l'outil de dégradé. Tracer dans la sélection la direction dans laquelle vous voulez que votre dégradé aie. Profit.

Dégradé depuis un outil de dessin :
Tout les outils de dessins(crayon, pinceau, etc.) peuvent faire des dégradés.
Il suffit de choisir la bonne dynamique de brosse.
